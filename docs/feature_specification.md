## 1. 기획실

### 1.1 캐릭터 카드

| 항목 | 스펙 |
|------|------|
| **기본 정보** | 이름, 나이, 성별, 직업, 종족 |
| **외모** | 키, 체형, 머리색, 눈색, 특징 (이미지 첨부) |
| **성격** | MBTI, 성격 태그, 장단점, 말투 샘플 |
| **배경** | 출생, 과거사, 트라우마, 목표 |
| **관계** | 다른 캐릭터와의 관계 링크 |

**기능:**
- 카드 복제 (변장/성장 버전)
- 검색 및 필터링
- AI 자동 생성 (키워드 기반)
- 이미지 생성 연동 (선택)

---

### 1.2 세계관 설정집

| 카테고리 | 포함 항목 |
|----------|----------|
| **지리** | 지도, 지역, 기후, 지형 |
| **역사** | 연표, 주요 사건, 시대 구분 |
| **사회** | 국가, 정치, 경제, 문화 |
| **마법/기술** | 체계, 규칙, 한계, 용어 |
| **종족** | 특징, 문화, 관계 |
| **용어집** | 고유 명사 사전 |

**기능:**
- 계층 구조 (대분류 > 중분류 > 항목)
- 크로스 레퍼런스 링크
- 집필 중 팝업 참조
- AI 기반 일관성 체크

---

### 1.3 관계도 에디터

```
시각적 관계도
┌─────────────────────────────────┐
│  [주인공] ──── 연인 ────► [히로인]  │
│      │                      │     │
│    라이벌                 친구    │
│      ▼                      ▼     │
│  [악역]  ◄──── 복수 ────  [조력자] │
└─────────────────────────────────┘
```

| 기능 | 설명 |
|------|------|
| **노드 추가** | 드래그로 캐릭터 배치 |
| **관계선** | 유형 선택 (가족, 연인, 적대 등) |
| **방향성** | 일방/양방 관계 표시 |
| **레이블** | 관계 설명 텍스트 |
| **그룹** | 세력/조직별 묶기 |
| **타임라인** | 시점별 관계 변화 추적 |

---

### 1.4 플롯 타임라인

| 뷰 모드 | 설명 |
|---------|------|
| **리스트** | 사건 목록 형태 |
| **타임라인** | 시간순 가로 배치 |
| **3막 구조** | 기승전결 프레임워크 |
| **스노우플레이크** | 확장형 플롯 구조 |

**복선 관리:**
- 미회수 복선 알림
- 복선 ↔ 회수 연결
- 색상 코딩

## 2. 집필실

### 2.1 집중모드 에디터

| 기능 | 스펙 |
|------|------|
| **디스트랙션 프리** | 전체화면, 메뉴 숨김 |
| **다크/라이트 모드** | 시스템 연동 또는 수동 |
| **폰트 설정** | 글꼴, 크기, 줄간격 조절 |
| **타자기 모드** | 작성 중인 줄 화면 중앙 고정 |
| **사운드스케이프** | 배경음 (비, 카페 등) 선택 |

---

### 2.2 씬/챕터 관리

```
프로젝트 구조
├── 시리즈 1: "드래곤의 후예"
│   ├── 1부: 각성
│   │   ├── 1장: 평범한 일상
│   │   │   ├── 씬 1: 아침 등교
│   │   │   ├── 씬 2: 교실 사건
│   │   │   └── 씬 3: 각성의 순간
│   │   └── 2장: 첫 번째 시련
│   └── 2부: 성장
└── 시리즈 2: "외전집"
```

| 기능 | 설명 |
|------|------|
| **드래그 정렬** | 씬/챕터 순서 변경 |
| **상태 표시** | 초안/수정중/완료 |
| **글자 수** | 씬/챕터/전체 카운트 |
| **메모** | 씬별 작가 노트 |
| **스냅샷** | 특정 시점 버전 저장 |

---

### 2.3 목표 추적기

| 목표 유형 | 설정 |
|----------|------|
| **일일 목표** | 글자 수 / 시간 |
| **주간 목표** | 글자 수 / 회차 |
| **프로젝트 목표** | 총 글자 수 / 완료일 |

**시각화:**
- 진행률 바
- 캘린더 히트맵
- 연속 달성 스트릭
- 통계 그래프

## 3. AI 어시스턴트

### 3.1 설정 검색 (자연어)

**입력 예시:**
```
"주인공이 처음 검을 얻은 장면"
"히로인과 악역이 처음 만난 회차"
"마법 체계의 5레벨 한계"
```

**출력:**
- 관련 씬/설정 목록
- 해당 위치로 바로 이동
- 관련도 순 정렬

---

### 3.2 이어쓰기 제안

| 모드 | 설명 |
|------|------|
| **문장 완성** | 현재 문장 이어쓰기 |
| **문단 생성** | 다음 문단 제안 (3개 옵션) |
| **대사 생성** | 캐릭터 성격 기반 대사 |
| **묘사 확장** | 배경/감정 묘사 추가 |

---

### 3.3 문체 변환

| 변환 유형 | 예시 |
|----------|------|
| **톤** | 유머 ↔ 진지, 가벼움 ↔ 무거움 |
| **시점** | 1인칭 ↔ 3인칭 |
| **문장 길이** | 간결 ↔ 만연체 |
| **장르 스타일** | 로판풍, 무협풍, 현대풍 |

---

### 3.4 캐릭터 시뮬레이션

```
┌─────────────────────────────────┐
│ 캐릭터: 레온 (츤데레 기사)         │
├─────────────────────────────────┤
│ 작가: "공주가 위험에 처했을 때     │
│       어떻게 반응할까?"            │
│                                  │
│ 레온: "흥, 내가 왜... 근데        │
│       혹시 어디 다친 곳은 없어?"  │
└─────────────────────────────────┘
```

- 캐릭터 카드 기반 성격 반영
- 대화/상황 시뮬레이션
- 복수 캐릭터 대화 생성

## 4. 스탯 매니저

### 4.1 상태창 템플릿

**남성향 RPG:**
```
┌─────────────────────────┐
│ 이름: 강민혁 │ Lv.27   │
│ 직업: 검투사           │
├─────────────────────────┤
│ 힘: 89  │ 민첩: 72     │
│ 지능: 45 │ 생명력: 156 │
└─────────────────────────┘
```

**여성향 로판:**
```
┌─────────────────────────┐
│ 이름: 아이린 베르티아   │
├─────────────────────────┤
│ ♥ 황태자: 45/100       │
│ ♥ 기사단장: 72/100     │
│ 평판: 68 │ 재력: 89    │
└─────────────────────────┘
```

### 4.2 자동 계산 & 일관성 체크

| 기능 | 설명 |
|------|------|
| **레벨업 연산** | 경험치/스탯 자동 증가 |
| **스킬 성장** | 사용 횟수 기반 레벨업 |
| **불일치 감지** | 에피소드별 스탯 추적, 오류 알림 |

## 7. 🚀 혁신 기능 (경쟁사 無)

### 7.1 🎙️ 음성 집필 모드

| 항목 | 스펙 |
|------|------|
| **엔진** | Whisper / Clova STT |
| **정확도** | 95%+ (웹소설 용어 커스텀 학습) |
| **후처리** | 자동 문장부호, 오타 교정 |
| **활용** | 이동 중 집필, 손목 부담 감소, 시각장애 지원 |

---

### 7.2 👥 베타리더 시스템

| 기능 | 설명 |
|------|------|
| **리더 프로필** | 선호 장르, 독서량, 피드백 스타일 |
| **매칭 알고리즘** | 작품 장르 ↔ 리더 취향 매칭 |
| **피드백 폼** | 몰입도, 페이싱, 캐릭터 + 자유 코멘트 |
| **리워드** | 포인트 적립 → 프리미엄 기능 교환 |

---

### 7.3 📊 독자 반응 예측 AI

| 분석 항목 | 지표 |
|----------|------|
| **템포 분석** | 씬별 속도감, 긴장-이완 곡선 |
| **클리프행어** | 회차 마무리 후킹 점수 |
| **설정 밀도** | 정보량 vs 스토리 비율 |

---

### 7.4 🎨 웹툰 스토리보드 연동

| 변환 | 설명 |
|------|------|
| 씬 → 패널 | 소설 씬을 웹툰 패널로 자동 분할 |
| 대사 추출 | 말풍선용 텍스트 분리 |
| 지문 → 연출 노트 | 그림 작가용 지시사항 |

---

### 7.5 🌍 다국어 번역
